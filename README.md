# Website

Official repository for [jonaseveraert.be](https://jonaseveraert.be).
Feel free to use any code for your own projects.

## Installation

### macOS

```sh
brew install libmagic sqlite3
```

### Dependencies

```sh
bundle install
bun install
ruby init.rb
```

### Running

```sh
ruby src/main.rb
```

## Running on FreeBSD

```sh
sh pot.sh
```
